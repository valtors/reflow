Framework support

One Library. Every Framework.

From hooks and composables to signals, stores, and controllers, reflow feels native in every modern UI stack.

React

Hooks 
const bp = useBreakpoint()
const desktopNav = bp.above('lg')

return desktopNav ? <Sidebar /> : null

Vue

Composable 
const bp = useBreakpoint()
const state = reactive({ desktop: bp.above('lg') })

return { state }

Svelte

Store 
const bp = breakpoint()

{#if $bp === 'lg'}
  <Sidebar />
{/if}

Solid

Signals 
const bp = useBreakpoint()

<Show when={bp.above('lg')()}>
  <Sidebar />
</Show>

Angular

Signals 
readonly bp = breakpointSignal()
readonly desktop = this.bp.above('lg')

protected sidebar = true

Lit

Controller 
private bp = new BreakpointController(this)

render() {
  return html`<app-shell ?desktop=${this.bp.above('lg')}></app-shell>`
}
