# reflow landing page

Static landing page for `reflow` built as a single `index.html` file with Tailwind CSS via CDN and vanilla JavaScript.

## Deploy

### Vercel
1. Create a new project and point it at this `website/` directory.
2. Framework preset: **Other**.
3. Build command: leave empty.
4. Output directory: `.`
5. Deploy.

### Netlify
1. Create a new site from your repo or drag-and-drop the `website/` folder.
2. Build command: leave empty.
3. Publish directory: `website`
4. Deploy.

## Local preview
Open `index.html` directly in a browser, or serve the folder with any static file server.
