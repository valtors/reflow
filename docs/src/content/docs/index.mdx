## Why reflow?

Responsive code usually means stitching together a media query hook, viewport listener, typography calculator, container helper, and SSR workaround. reflow replaces that pile with one typed toolkit.

- **SSR-safe by default** with `useSyncExternalStore`-powered React hooks and server breakpoint helpers.
- **Typed everywhere** from `createBreakpoints()` to `<Show />` and `useResponsiveValue()`.
- **Framework-agnostic** core with optional React, Vue, and Svelte adapters.
- **Modern CSS support** for `clamp()`, container queries, safe-area insets, and dynamic viewport units.
