# Installation

## Package manager commands

```bash
npm install reflow
```

```bash
pnpm add reflow
```

```bash
yarn add reflow
```

## Optional peer dependencies

The core package works on its own. Framework peers are only needed if you use the matching adapter:

- `react >= 17` for `reflow/react`
- `vue >= 3.3` for `reflow/vue`
- `svelte >= 4` for `reflow/svelte`

If you only use core, styles, server, testing, or Tailwind helpers, you do **not** need to install React, Vue, or Svelte.

```ts
import { createBreakpoints, mq, watchMedia } from "reflow";
import { ResponsiveProvider, useBreakpoint } from "reflow/react";

```

## Tree-shakeable entry points

| Import | What you get |
| :-- | :-- |
| `reflow` | Core breakpoints, media, viewport, container, preferences, pointer, DPR, safe area, and store APIs |
| `reflow/react` | React hooks and components |
| `reflow/vue` | Vue 3 composables |
| `reflow/svelte` | Svelte stores |
| `reflow/styles` | CSS helpers like `fluidClamp()` and `containerQuery()` |
| `reflow/server` | Client Hints and user-agent SSR helpers |
| `reflow/testing` | Test mocks for DOM environments |
| `reflow/tailwind` | Tailwind screen preset |

## Minimal core-only install

```ts
import { createBreakpoints, watchMedia, observeViewport } from "reflow";

const system = createBreakpoints({ mobile: 0, tablet: 768, desktop: 1024 } as const);

watchMedia(system.up("tablet")).subscribe((matches) => {
  console.log("tablet and up", matches);
});

observeViewport(({ width }) => {
  console.log("active breakpoint", system.resolve(width));
});
```
